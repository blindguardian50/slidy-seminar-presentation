Responsive Visualization Tools

Chart.js

  • Responsive visualization toolkit
  • Open-source library for creating chart via HTML canvas element
  • Offers eight base chart types
  • Rendering is highly performant even with large number of elements
  • Default configuration already leads to appealing results
  • Offers basic interactive features
  • Size of minified bundle: 66.6KB
  • Popularity: 62K stars on GitHub
  • Performance test: 57FPS with unchanged data, 53FPS with extended data set

Plotly.js

  • Responsive visualization toolkit
  • Open-source library for creating visualizations via SVGs
  • Support for over 40 different chart types
  • Built on top of famous visualization library D3
  • Default configuration not so straight forward
  • By default includes a toolbar for interacting with visualization
  • Elements can be made highly interactive
  • Size of minified bundle: 1.1MB
  • Popularity: 15.9K stars on GitHub
  • Performance test: 20FPS with unchanged data, 25FPS with extended data set

Chartist

  • Responsive visualization toolkit
  • Lightweight open-source library for creating visualizations via SVGs
  • Offers bar, line and pie charts as well as subtypes and variations
  • Offers a default style sheet
  • Elements can be made highly interactive
  • Titles cannot be added to SVGs (plugin for axis titles exists)
  • Scaling of elements only via container or media queries
  • Library size of minified bundle: 11.7KB
  • Popularity: 13.3K stars on GitHub
  • Performance test: 57FPS with unchanged data, 49FPS with extended data set

Highcharts

  • Responsive visualization toolkit
  • Source-available library for creating visualizations via SVGs
  • Free for non-commercial and educational purposes
  • Offers over 50 different visualization types
  • Offers own modules for dashboards, maps, stocks and gantt charts
  • Offers smooth default animations on initial load
  • Elements can be made highly interactive
  • Provides extensive documentation (with AI chatbot)
  • Size of minified bundle: 96.5KB
  • Popularity: 11.5K stars on GitHub
  • Performance test: 56FPS with unchanged data, 52FPS with extended data set

Hoffswell Visualization System

  • Responsive visualization system
  • Tool for designing visualizations in parallel
  • Modifications of one visualization are propagated down to others
  • Published in proceedings of the 2020 CHI Conference on Human Factors in Computing Systems
  • Tool not publicly available
  • Example gallery contains created visualizations and showcase videos

Power BI

  • Responsive visualization system
  • Set of services provided by microsoft
  • Enables collecting, analyzing, transforming data
  • Enables creating and collaborative sharing of visualizations and reports
  • Designer can change size of visualization on creation of report and visualization will adapt automatically
  • Different reports for mobile and desktop view can be created
  • Visualizations will not automatically adapt in published reports without further adjustments

MobileVisFixer

  • Visualization transformation tool
  • Automatically transforms existing SVG based graphics into mobile friendly ones
  • Uses reinforcement-learning-based approach
  • Published in IEEE Transactions on Visualization and Computer Graphics 2021
  • Tool not publicly available

Setlur Line Chart Resize Algorithm

  • Visualization transformation tool
  • Automatically resizes line charts to fit different size requirements
  • Implementation of algorithm via HTML, JavaScript and D3
  • Source code not publicly available
